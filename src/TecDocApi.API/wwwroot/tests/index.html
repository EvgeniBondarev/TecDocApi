TecDoc Search Test Bench

Страница для ручного тестирования поиска, проверки индексации, просмотра полного URL, тела запроса, сырого JSON и изображений без переходов по Swagger.

Поиск артикулов

Возвращает только данные из Elasticsearch без картинок. Для изображений используйте отдельную секцию ниже.

Файл изображения по имени

Проверка конкретного файла через supplierId и fileName.

Картинки по артикулу

Отдельный запрос в `S3Info` и S3 через `/api/Images/article-search`. Используйте после поиска артикула или независимо от него.

Глобальный поиск по S3

Отдельный поиск по всем папкам S3 через `/api/Images/s3-search`. Пример: `ALM2019YX` должен находить `ALM2019YX-6` и другие вариации.
